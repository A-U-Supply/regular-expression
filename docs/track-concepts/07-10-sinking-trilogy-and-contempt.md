# Tracks 07–10: Sinking Trilogy Finale + Contempt Opens

## Album Arc Position
- **07**: Sinking trilogy finale (with 05–06). Reverse structure.
- **08–10**: The album turns outward — contempt erupts in angular, seasick forms.

---

## Track 07: "shittiest"

**Concept:** The sinking trilogy finale. Where 05 built and 06 sagged, 07 starts at the climax and strips everything away. Fortissimo at bar one. Each instrument falls off one by one until only the pad is left, fading.

| Attribute | Value |
|-----------|-------|
| Key | A minor (natural minor) |
| Tempo | 90 BPM (slowest in album) |
| Time Sig | 4/4 |
| Bass | Electric Bass Finger (33) |
| Melody | Distortion Guitar (30) |
| Chords | Pad Warm (89) |
| Directive | Reverse Structure |

**Structure:**
1. 4 bars — all instruments fortissimo (velocity +20)
2. 4 bars — drums drop, rest continues at 90 velocity
3. 3 bars — melody drops, bass + pad remain
4. 2 bars — bass drops, pad alone fading
5. 1+ bar — single pad chord at velocity 30, trailing off

**Theory:**
- Progression: i–III–VII–iv (same family as the sinking trilogy)
- Pad Warm gives these chords an exhausted, waterlogged quality
- No re-entry — the tradition of resolution (from 05) is gone

## Track 08: "dumbass"

**Concept:** Angular, ugly 5/4 stumble. A 2-bar loop that mutates across four repetitions, degrading its own structure. Then anxious no-snare section, then blast, then silence.

| Attribute | Value |
|-----------|-------|
| Key | F# minor (Phrygian) |
| Tempo | 144 BPM |
| Time Sig | 5/4 |
| Bass | Electric Bass Pick (34) |
| Melody | Lead 1 Square (81) |
| Chords | Overdriven Guitar (29) |
| Directive | Loop That Breaks |

**Structure:**
1. 4 × 2-bar loops (~24s) — each rep mutates (pitch drift, timing shift, note dropouts)
2. 4 bars — no-snare anxious drums, stutter single note melody
3. 2 bars — blast beat, full power chord
4. Silence (ends with decay)

**Theory:**
- F# Phrygian: bII = G natural, the source of angular ugliness
- 5/4 creates a constant limp — bars never feel "finished"
- Lead Square tone gives the melody a synthetic, broken quality

## Track 09: "horrible"

**Concept:** Self-lacerating. The D1 drone (MIDI 26) creates constant subterranean dread. Everything builds FROM the drone — and collapses back to it.

| Attribute | Value |
|-----------|-------|
| Key | D minor (natural minor) |
| Tempo | 122 BPM |
| Time Sig | 4/4 |
| Bass | Electric Bass Finger (33) |
| Melody | Distortion Guitar (30) |
| Chords | Overdriven Guitar (29) |
| Directive | The Drone (D1=26 sustained) |

**Structure:**
1. 2 bars — drone alone (D1, velocity 42)
2. 4 bars — drums + bass enter (half-time feel)
3. 4 bars — chords + melody enter with cluster stab at bar 9
4. 2 bars — drop to drone + one sustained note (D4)
5. 2 bars — full re-entry fortissimo

**Theory:**
- D1 (MIDI 26) is below E1 — deeper, more cavernous than the 05/07 drone
- Cluster on D4 (center=62, width=2) at bar 9 is the album's most dissonant moment so far
- Ascending melody at re-entry contrasts with all previous descending contours

## Track 10: "awful"

**Concept:** Seasick 6/8. The waltz feel wobbles under increasing pressure until everything collapses to a single sustained B across all instruments, then erupts.

| Attribute | Value |
|-----------|-------|
| Key | B minor (Phrygian) |
| Tempo | 110 BPM |
| Time Sig | 6/8 |
| Bass | Electric Bass Finger (33) |
| Melody | Distortion Guitar (30) |
| Chords | Electric Guitar Clean (27) |
| Directive | Unison Collapse |

**Structure:**
1. 4 bars — clean waltz (half-time drums, clean guitar)
2. 4 bars — intensify (driving emo drums, distortion melody)
3. 2 bars — unison collapse on B (B1/B2/B3 sustained, floor tom pulse only)
4. 4 bars — blast eruption, all instruments at peak

**Theory:**
- 6/8 compound meter creates a rolling nausea that 4/4 can't
- B Phrygian bII = C natural, which in 6/8 feels like a stumble
- The unison collapse strips harmonic content entirely — pure pitch mass
