# Tracks 11–13: The Conjugation Trilogy

## Album Arc Position
Three conjugations of the same phrase (piss off / pissed off / pissing off), each using recognizably shared melodic material but radically different in key, tempo, and structure. The listener may not consciously hear the connection — but should feel it.

---

## Shared Material

All three tracks derive from `TRILOGY_MELODY_PITCHES` defined in track 11:
```
E4, D4, C4, A3, B3, G3, A3, B3, C4, D4, E4, F#4
```
A descending then ascending shape — frustration stated, circling, not resolved.

- **Track 12** transposes this up a fifth to Bm (+ 7 semitones), faster, displaced
- **Track 13** transposes it up a fourth to Am (+ 5 semitones), slower, 7/8, stripped

---

## Track 11: "piss off"

**Concept:** The base form. Countdown sections get shorter as intensity grows. Each section is hotter than the last. Defines the core melodic material of the trilogy.

| Attribute | Value |
|-----------|-------|
| Key | E minor (natural minor) |
| Tempo | 132 BPM |
| Time Sig | 4/4 |
| Bass | Electric Bass Pick (34) |
| Melody | Distortion Guitar (30) |
| Chords | Overdriven Guitar (29) |
| Directive | The Countdown (8→4→2→1) |

**Structure:**
1. 8 bars — driving emo, full band, melody from TRILOGY_MELODY_PITCHES
2. 4 bars — no-snare drums, louder, melody repeats octave higher
3. 2 bars — blast beat, stutter melody
4. 1 bar — single crash + bass root + melody note at max

## Track 12: "pissed off"

**Concept:** The past-tense form. Same material, shifted to Bm, faster (148bpm), with the melody rhythmically displaced so it lands wrong against the rhythm section.

| Attribute | Value |
|-----------|-------|
| Key | B minor (natural minor) |
| Tempo | 148 BPM |
| Time Sig | 4/4 |
| Bass | Electric Bass Pick (34) |
| Melody | Distortion Guitar (30) |
| Chords | Overdriven Guitar (29) |
| Directive | Rhythmic Displacement (melody displaced 2 eighths) |

**Structure:**
1. 8 bars — full band, melody from BM_MELODY_PITCHES displaced by 2 eighths
2. 4 bars — blast beat, melody at octave up, displaced by 1 eighth
3. 2 bars — final crash + chord + single melody note

**Theory:**
- Transposition to Bm is a fifth up — feels "higher stakes" without changing shape
- Displacement by 2 eighths (one full beat) makes the melody feel like it arrives a beat late

## Track 13: "pissing off"

**Concept:** The present-progressive form. The same material inverted structurally — starts full, strips away. Slower, 7/8, adding rhythmic instability to the melodic familiarity.

| Attribute | Value |
|-----------|-------|
| Key | A minor (natural minor) |
| Tempo | 116 BPM |
| Time Sig | 7/8 |
| Bass | Electric Bass Finger (33) |
| Melody | Lead 1 Square (81) |
| Chords | Overdriven Guitar (29) |
| Directive | Reverse Structure |

**Structure:**
1. 5 bars — full band fortissimo, melody from AM_MELODY_PITCHES
2. 4 bars — drums drop, others continue, melody with broken trail
3. 3 bars — chords drop, bass + floor tom + single fading melody note

**Theory:**
- Am transposition is a fourth up from Em — "lower" emotionally despite similar pitch range
- 7/8 makes the familiar melody stumble — the grammar of frustration has become irregular
- Lead Square timbre makes the same notes feel synthetic, exhausted
