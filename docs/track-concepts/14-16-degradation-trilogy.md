# Tracks 14–16: The Degradation Trilogy

## Album Arc Position
Three forms of the phrase "piece of ___". Starts aggressive (shit), becomes defeated (crap), hollows out (junk). A structural descent from fury to near-silence.

---

## Track 14: "piece of shit"

**Concept:** Aggressive. The degradation trilogy starts at maximum attack — blast from bar one, then a sudden collapse to solo melody, then re-entry even harder.

| Attribute | Value |
|-----------|-------|
| Key | D minor (Phrygian) |
| Tempo | 155 BPM |
| Time Sig | 4/4 |
| Bass | Electric Bass Pick (34) |
| Melody | Distortion Guitar (30) |
| Chords | Overdriven Guitar (29) |
| Directive | The Collapse |

**Structure:**
1. 6 bars — blast beat + driving emo, full fury
2. 3 bars — collapse to solo melody only (velocity 70, ascending, low octave)
3. 4 bars — re-entry fortissimo, spike at re-entry, cluster hit

**Theory:**
- Dm Phrygian: Eb (bII) creates maximum harmonic wrongness
- Spike at re-entry (base=D4, interval=min7) is the climactic moment
- Solo melody section has bass note at velocity 45 — present but barely

## Track 15: "piece of crap"

**Concept:** Defeated. The waltz feel of 3/4 at 120bpm suggests something that should be graceful, but the loop degrades with each repetition until there's nothing left.

| Attribute | Value |
|-----------|-------|
| Key | A minor (natural minor) |
| Tempo | 120 BPM |
| Time Sig | 3/4 |
| Bass | Electric Bass Finger (33) |
| Melody | Distortion Guitar (30) |
| Chords | Electric Guitar Clean (27) |
| Directive | Loop That Breaks |

**Structure:**
1. 4 × 2-bar loops — drums: driving → half-time → floor tom → just one kick
2. Each loop: melody degrades (pitch drift, timing jitter, note dropout)
3. Each loop: chords fade (85→70→55→40 velocity)
4. Final 4 bars — drums gone, counter melody bass, broken trail melody

**Theory:**
- Clean guitar for chords emphasizes the "graceful failure" quality
- Loop degradation maps 3/4's natural circular feeling onto entropy

## Track 16: "piece of junk"

**Concept:** Hollowed out. Em Dorian. E1 drone (28). Parts appear so sparse they might not be there. The Pad Warm melody barely registers. The track exists in the space between sounds.

| Attribute | Value |
|-----------|-------|
| Key | E minor (Dorian) |
| Tempo | 95 BPM |
| Time Sig | 4/4 |
| Bass | Electric Bass Finger (33) |
| Melody | Pad Warm (89) |
| Chords | Electric Guitar Clean (27) |
| Directive | The Drone (E1=28) |

**Structure:**
1. Bars 1-2 — drone alone
2. Bars 3-4 — single floor tom hit + one hat per bar
3. Bars 5-8 — sparse chords (60% chance per bar), slow Pad Warm phrase (density=0.25)
4. Bars 9-12 — half-time drums quiet, sparse chords, slow ascending melody
5. Final bars — just drone + one held chord at velocity 25

**Theory:**
- Dorian mode (F# natural, C# natural) adds 2 non-minor notes — slight shimmer in the void
- Pad Warm for melody makes every note a dying breath
- The drone outlasts everything — it was there first, it's there last
